# АналогJira — приложения для управления программными проектами

## Описание проекта

АналогJira — это веб-приложение для управления проектами, поддерживающее методологии Agile/Scrum и Kanban. Позволяет создавать задачи, управлять спринтами, отслеживать прогресс через диаграммы сгорания задач и получать уведомления в реальном времени.

## Ключевые функции

- Kanban-доска с перетаскиванием (Drag-and-Drop)
- Планирование спринтов и управление бэклогом
- Диаграмма сгорания задач (Burndown Chart)
- Система комментариев и вложений
- Уведомления в реальном времени (WebSocket)
- Отчёты по скорости команды (Velocity)

## Используемые технологии

| Компонент | Технология |
|-----------|------------|
| Frontend | React (Drag-and-Drop) |
| Backend | Node.js / FastAPI |
| База данных | MongoDB |
| Реальное время | WebSocket, JWT |
| Управление версиями | Git, GitHub |
| Управление проектом | YouTrack |
| CI/CD | GitHub Actions |
| Тестирование | Jest, Postman, Selenium |
